### Setup

1) Make sure you have gulp installed globally:

```bash
$ npm install -g gulp
```

2) Install dependencies:

```bash
$ npm install
```

### Usage

**tl;dr** Run `gulp live` and get to work.

| Command      | Description                                             |
| ------------ | ------------------------------------------------------- |
| `gulp live` | Start server on port 3000 & live-update as files change |
| `gulp publish` | Compile everything, fingerprint assets, and copy to `/publish`. |
